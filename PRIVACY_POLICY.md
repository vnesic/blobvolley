# Privacy Policy for Blob Volley

**Last updated:** June 14, 2026

## Overview

Blob Volley is a volleyball game with local multiplayer support. This privacy policy explains how the app handles user data.

## Data Collection

**This app does not collect, store, or transmit any personal data.**

We do not collect:
- Personal information
- Usage analytics
- Crash reports
- Advertising identifiers
- Location data

## Network Permissions

The app requests the following network permissions solely for local LAN multiplayer functionality:

- `INTERNET` - Required for local network communication
- `ACCESS_NETWORK_STATE` - To detect network availability
- `ACCESS_WIFI_STATE` - To discover other players on WiFi
- `CHANGE_WIFI_MULTICAST_STATE` - For LAN game discovery

**No data is sent to external servers.** All network communication occurs only between devices on your local network during multiplayer games.

## Third-Party Services

This app does not use any third-party services, analytics, or advertising SDKs.

## Children's Privacy

This app does not collect any personal information from anyone, including children under 13.
